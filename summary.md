

## 数据结构

### 一维

* 基础：数组 array(string)、链表 linked list
* 高级：栈 stack、队列 queue、双端队列 deque、集合 set、映射 map

### 二维

* 基础：树 tree、图 graph
* 高级：二叉搜索树 BST、堆 heap、并查集 disjoint set、字典树 Trie

### 特殊

* 位数算 bitwise
  * 清零
  * 获取第 n 位值
  * 第 n 位幂值
  * 第 n 位置1
  * 第 n 位置0
* 布隆过滤器 BloomFilter
* LRU Cache：Hash Table + Double LinkedList
  * O(1) 查询
  * O(1) 修改、更新
